# New Project
This project is a social media prototype feature that allows users to like a post by double-clicking on it, just like popular social media apps.

🚀 Features
Double-Click to Like – Users can like the post by double-clicking on the image/post.

Heart Animation – A heart icon appears briefly to give visual feedback.

Like Counter – Keeps track of total likes dynamically.

Interactive & Lightweight – Built using HTML, CSS, and JavaScript only.

🛠️ Technologies Used
HTML5 – For structuring the post

CSS3 – For styling and heart animation effects

JavaScript (DOM) – To detect double-click events and update likes
